6.html is "tag based" markup language

	EX: <table></table>
		<p>

7.There are 2 types of Tag's
	
	1) Paired Tag / container tag's
	
		=> it contains both opening and closing tags
			ex: <html></html>
			    <head></head>
			    <body></body>
	
	2) unpaired Tag / non-container tag's

		=> it contains only opening tag
			ex: <pr> , <hr> , <img>, <br>

8. Each Tag is enclosed with <> Tags , closed tag contains forward slash</>
